【GraphQLとは】
GraphQLはFacebook社を起源とするAPIのクエリ言語です。
１つのAPIエンドポイントから必要とするデータを全て取得可能なため、
各リソースごとにリクエストをするREST APIと比較して効率的な開発をすることが出来ます。
